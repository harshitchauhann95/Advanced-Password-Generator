A Python-based GUI application that generates secure and customizable passwords. Built with **Tkinter**, this tool helps you create strong passwords with ease, copy them to your clipboard, and check their strength instantly.

## Features

- Generate passwords with:
  - Letters (uppercase & lowercase)
  - Digits
  - Symbols
- Exclude visually similar characters (O, 0, l, 1, I) to avoid confusion.
- Option to **ensure at least one character** from selected types is included.
- Adjustable **password length** with a slider (6‚Äì40 characters).
- **Password strength indicator**: Very Weak ‚Üí Very Strong.
- **Copy password to clipboard** with a single click.
- User-friendly instructions built into the app.

## Installation

1. Make sure you have **Python 3.x** installed.
2. Install Tkinter (usually comes with Python) if not installed: pip install tk
3. Clone this repository or download the files.

## Usage

1. Run the app: python password_generator.py
2. Adjust the password length using the slider.
3. Select the character types you want to include.
4. (Optional) Exclude similar characters to make the password more readable.
5. Click **Generate Password** to create a new password.
6. Click **Copy Password** to copy it to your clipboard.
7. Check the password strength below the input field.
